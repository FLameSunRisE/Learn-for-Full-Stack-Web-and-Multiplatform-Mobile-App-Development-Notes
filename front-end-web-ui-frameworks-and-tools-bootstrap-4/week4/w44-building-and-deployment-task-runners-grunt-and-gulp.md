# Grunt Part 1

* Installing Grunt globally
  ```
  npm install -g grunt-cli
  ```

* install the Grunt CLI in project
  ```
  npm install grunt --save-dev
  ```
* Creating a Grunt File

  * Gruntfile.js:

    ```js
    'use strict';

    module.exports = function (grunt) {
      // Define the configuration for all the tasks
      grunt.initConfig({

      });
    };
    ```
